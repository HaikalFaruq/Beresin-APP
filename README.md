## 📂 Installation

```bash
# Clone repository
git clone https://github.com/HaikalFaruq/Beresin-APP.git

# Masuk ke folder project
cd Beresin-APP

# Install dependencies
npm install

# Jalankan di mode development
npm run dev

# Build untuk production
npm run build
